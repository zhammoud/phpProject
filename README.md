## Installation

Run:
<p>- composer install</p>
<p>- npm install</p>
<p>- npm run dev</p>
<br>
- make sure the database configuration are as follow:
<pre>
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=phpproject
DB_USERNAME=root
DB_PASSWORD=toor
</pre>

<p>- php artisan migrate</p>
<p>- php artisan serve</p>
